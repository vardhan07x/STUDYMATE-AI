# StudyMate AI Agent

A simple AI agent to help students stay productive during competitive exam preparation.

## Install in Kaggle

Run this:

pip install git+https://github.com/vardhan07x/studymate-agent.git

## Use the agent

from agent import StudyMateAgent

agent = StudyMateAgent("YOUR_API_KEY")
print(agent.reply("Create a timetable for GATE exam preparation"))
